# doc2json
Service for parsing documents and returning them in JSON format.

## Supported formats
- doc
- docx
- pdf
- rtf
- txt

## Installation
```bash
python3 -m venv venv
```
```bash
source venv/bin/activate
```
```bash
pip install -r requirements.txt
```
